# Qleet homebrew-tap
Qleet Homebrew formulae, casks and/or external commands

## How do I install these formulae?

```bash
brew tap qleet/tap
brew install qleet/tap/<formula>
```

For example:

```bash
brew tap qleet/tap
brew install qleet/tap/qleetctl
```
